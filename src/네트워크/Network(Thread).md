# 네트워크(스레드)

### 프로그램, 프로세스, 스레드

##### 프로그램
- 어떤 문제를 해결하기 위하여 그 처리 방법과 순서를 기술하여 컴퓨터에 주어지는 일련의 명령문 집합체

##### 프로세스
- 컴퓨터 내에서 실행중인 프로그램

##### 스레드
- 다중 작업 환경에서 운영 체제가 여러 프로그램을 병행하여 처리할 때 효율적인 처리를 위하여 분할한 처리 단위


### 멀티 스레드

##### 멀티프로세스
- 여러 흐름이 동시에 진행
- 각 프로세스는 독립적으로 실행
- 각각 별개의 메모리를 차지
- 전환 속도가 상대적으로 느림

##### 멀티 스레드
- 여러 흐름이 동시에 진행
- 프로세스 내의 메모리를 공유
- 전환 속도가 상대적으로 빠름


### 임계구역
- 둘 이상의 스레드가 동시에 접근해서는 안되는 공유 자원을 접근 하는 코드의 일부

###### 임계구역을 해결하기 위한 3가지 요구 조건
1. 상호배제 : 한 프로세스만 임계구역에 진입
2. 진행 : 임계 구역에서 작업 중인 프로세스가 없다면 임계 구역으로 진입하려는 프로세스 중 하나를 적절히 선택하여 임계구역에 진입
3. 제한된 대기 : 한 프로세스가 임계구역 진입 요청 후 기다리는 데 한계가 있음
